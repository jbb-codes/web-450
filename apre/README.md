# APRE - Agent Performance Reporting Engine

A full-stack MEAN application for sales and agent performance reporting.

## Project Structure

```
apre/
├── apre-client/   # Angular 18 frontend
└── apre-server/   # Node/Express REST API
```

## Getting Started

### Server

```bash
cd apre-server
npm install
npm start
```

### Client

```bash
cd apre-client
npm install
ng serve
```
